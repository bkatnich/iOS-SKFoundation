## Requirements

- Xcode 9.0+
- iOS 11.0+
- Swift 4.0


## Installation

### CocoaPods

The preferred installation method is with [CocoaPods](https://cocoapods.org). Add the following to your `Podfile`:

```ruby
pod 'SandKattFoundation'
```
